# kafka-ui-lib

 [kafka-ui-lib](http://github.com/Landoop/kafka-ui-lib) is a AngularJS library to interact with Kafka end-points i.e. Rest APIs

## Installation

```bash
$ npm install kafka-ui-lib
```

## Features

  - schema-registry REST APIs
  - common http methods

## Publishing

Build and publish:

    npm run build-prod
    npm login (landoop/pass)
    npm publish
